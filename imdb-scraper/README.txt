IMDB searching Movies Project

I Build a web scraper with Node.js, "cheerio", "node-fetch", "prompt" packages on IMDB website, I used "cheerio" package to get data from html tags like baked-in DOM, I used "node-fetch" package to get html data from url website and i used "prompt" package to beautiful command-line prompt and get data from client.
